# FullStack CRUD App

## Descripción
Este proyecto implementa un formulario básico con operaciones CRUD usando:
- **Backend**: NestJS + TypeORM + MongoDB
- **Frontend**: Vue.js + TailwindCSS + PrimeVue + Pinia
- **DevOps**: Docker + Docker Compose + CI/CD con GitHub Actions

## Requisitos
- Node.js v20+
- Docker
- MongoDB
